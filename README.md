# Sistema de Reserva de Hotel
Esse projeto foi desenvolvido em Python com foco também em programação orientada a objetos.
A proposta do sistema é simular uma reserva de hotel de forma simples, trabalhando com hóspedes, quartos e reservas.

## Funcionalidades
  Cadastro de hóspede; cadastro de quarto; realização de reserva; cálculo do valor total da hospedagem; cancelamento de reserva e exibição de resumo da reserva

## Estrutura do projeto
- `hospede.py` : classe responsável pelos dados do hóspede
- `quarto.py` : classe responsável pelos dados do quarto
- `reserva.py` : classe responsável pelas regras da reserva
- `main.py` : arquivo principal para testes e execução

## Conceitos aplicados
  Classes e objetos; encapsulamento, associação entre objetos, métodos com regra de negócio, separação em arquivos.

## Como executar
1. Abra o projeto no VS Code
2. Execute o arquivo `main.py`

## Tecnologias utilizadas
- Python
